# Judgment Pack Specification `0.1.0-draft`

This release publishes the first research preview of the Judgment Pack Specification (JPS) under
the immutable tag `v0.1.0-draft`.

It is intended for design review, synthetic pack authoring, and carrier, structural, and semantic
conformance experiments. It is not an industry standard, carries no compatibility guarantee, and
must not be represented as sufficient for consequential production decisions.

## Included

- Core prose and a JSON Schema Draft 2020-12 structural schema.
- Three unrelated synthetic examples.
- A versioned 47-case carrier, structural, semantic, and capability corpus.
- A short external-user testing exercise and feedback template.
- Automated repository integrity and corpus checks.
- Versioned `.tar.gz` and `.zip` artifact bundles, SHA-256 checksums, and GitHub build-provenance
  attestations.

## Compatibility

This is the first draft release. Later `0.x` releases may be incompatible in reader, writer, or
semantic behavior.

## Important limitation

Evaluator conformance is not defined. The condition and resolution material records experimental
design intent; it is not a portable runtime contract. Conformance does not establish truth,
authorization, safety, or operational fitness.

See `TESTING.md`, `VERSIONING.md`, and `conformance/README.md` in this release for details.
